# OrderedJson


### Usage

​	step 1.  Define a static class

​	step 2.  Define a class which extends **OJContext**

​	step 3.  Instantiate **OJParser** through the static class

​	step 4.  Call `OJParser.Parse` using a string(script),  got **IOJMethod**.

​	step 5.  Invoke that **IOJMethod** using the **OJContext** defined in step2.
